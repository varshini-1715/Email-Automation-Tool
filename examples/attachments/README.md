# Example Attachments

This directory is reserved for safe example attachment files.

No sample PDF is included because an empty file is not a valid PDF.

To test attachments:

1. Place a real non-confidential file in this directory.
2. Run `python main.py`.
3. Select an email workflow that supports attachments.
4. Enter the file path when prompted.

Do not commit confidential documents, customer files, or private data.
